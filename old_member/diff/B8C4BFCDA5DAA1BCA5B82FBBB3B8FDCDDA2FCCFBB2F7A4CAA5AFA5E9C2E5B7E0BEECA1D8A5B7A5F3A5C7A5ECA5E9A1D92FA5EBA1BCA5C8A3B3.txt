=[[個人ページ/山口遥/愉快なクラ代劇場『シンデレラ』]]
=
=お父さんのその問いにシンデレラは困ってしまいました。いまここであの靴を履きに行けば、
=
=ここであの夜王子と一緒に踊っていたのは私ですと言いに行けば。
=
=シンデレラは王子と結婚できるかもしれません。でもほんとにそれでいいの？
=
=とシンデレラの頭の中はごちゃごちゃです。
=
=「わ、私裏庭の掃除してきますっ」
=
=シンデレラはそう言って逃げるようにその場から去りました。そして裏庭の馬小屋に行き、
=
=舞踏会の日にシンデレラをお城まで連れて行ってくれた馬を撫でました。
=
=「お馬さん、私よくわからなくなって飛び出してきちゃったわ。あの時の王子様とのダンスは
=
=楽しかったんだけど、いきなり結婚って言われても…。なんてお馬さんに話しても
=
=どうしようもないわね、私ったら何しているのかしら…」
=
=シンデレラは馬さんを撫でながらため息をつきました。すると、どこからか
=
=聞いたことのある声がしました。
=
=「シンデレラ、君は馬鹿な人だなあ…あのまま靴を履いていれば幸せになれたかもしれないのに…
=
=わざわざここにくるなんて…」
=
=「この声…もしかしてお馬さん？」
=
=シンデレラはあの舞踏会の夜を思い出しました。シンデレラが舞踏会に心を躍らせている中、
=
=魔法の解ける時間を忘れないようにと、優しく言ってくれたその声、
=
=馬の声とさっきの声が一緒だったのです。
=
=「まったく、君はせわしないな。ここではいつも家事を押し付けられて、
=
=お城では王子様と出会えたと思ったら時間ぎりぎりに戻ってきて。
=
=王子ともう一度対面できるチャンスを自らふいにしてしまって。シンデレラ、君はそれでいいのかい？」
=
=馬は優しくシンデレラに語りかけました。シンデレラはただ馬を撫で続けます。
=
=「今ならまだ間に合うかもしれないよ？行かなくていいのかい？」
=
=その言葉にシンデレラは悩みました。戻ったら王子と結ばれることができるかもしれません。
=
=継母や意地悪な姉たちにもう家事を押し付けられることもないかもしれません。
=
=でもシンデレラは馬にこう告げました。
=「私ね、まだ結婚なんて考えられないの。王子様は確かに素敵だったけど、
=
=たった一回しかお話したことないのよ？それにね、私なんだかんだ言ってこの家にまだいたいのよ。
=
=そりゃお母様やお姉さま方は意地悪なさるし、家事は大変。
=
=でも大好きなお父様と一緒に入れるのはこの家だし、それにここから出て行ってしまったら、
=
=あなたともお話しできないしね。だから、お嫁に行くのはもう少し先にするわ。」
=
=シンデレラのこの言葉を聞いた馬は驚きましたが、その後くすりと笑いました。
=
=馬なのでくすりと笑ったかは微妙ですが。
=
=「シンデレラ、君はお馬鹿さんだよ。すごいチャンスを不意にしたんだからね。
=
=でも君らしい。これからも僕の世話をよろしく頼むよ」
=
=「うふふ、はーい。ところでお馬さんだからさっきから私のことしきりに馬鹿っていうの？
=
=狙ってるの？あれ、どうしてお馬さん黙ってるの？ねえ？」
=
=こうしてシンデレラは今までと同じ生活に戻るのでした。
=
=王子がその後花嫁を迎えられたのかは、シンデレラの知るところではありません。
