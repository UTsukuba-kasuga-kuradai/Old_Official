=
=シンデレラのお父さんはシンデレラのお母さんが亡くなったあと、 二人の娘がいる女性と結ばれましたが、~
=
=その女性はひどい猫かぶりで とんでもなく意地悪でした。そして残念なことにその二人の娘もひどくひねくれた 娘たちで、~

=意地悪な継母と姉たちはシンデレラのお父さんのいない時を狙ってシンデレラを ひどくいじめるのでした。~
=
=Twitterの鍵つきアカウントでこぞって悪口を呟くなどと、あまりにも継母と姉たちが巧妙な手口でシンデレラをいじめるので、~
=
=シンデレラのお父さんはシンデレラがいじめられていることに気がついていませんでした。 女って怖い。~
=
=床を磨くのも洗濯するのも、あらゆる家事はシンデレラに押し付けられていました。~

=「私なんかが舞踏会に行っても踊れないし、野菜も食べれないし、周りの方にご迷惑を おかけしてしまうわ･･･~

=と言って家の仕事をし始めました。 すると突然背の高い、（たぶん）魔法使い（っぽい人）が現れ、~

=シンデレラの前に仁王立ちしました。~

=「………どちら様でしょう？」~

=すぐに自分の状況に気づいて悲しむ顔をしたシンデレラの肩に、魔法使いは手を置きました。~

=「私が魔法ですべて叶えてあげよう。素敵なドレスも、お城まで君を連れて行ってくれる馬車も私が魔法で出そうじゃないか！」~
=
=そう言うと魔法使いは次の呪文を唱えました。~
=
=「ニンジン・タマネギ・ジャガイモ・オニク・カレーコ・タマニハカボチャモイレヨウ・カクシアジハ・リンゴ・コーヒー・ソノタモロモロ！」~
=
=するとなんという事でしょう。シンデレラは普段着ているみすぼらしい服ではなく、綺麗なドレスをまとっていて、家にあったカボチャは馬車になりました。~
=
=「まあ、なんてことなの？」~
=
=困惑するシンデレラに、魔法使いはこう告げました。~
=
=「この魔法はずっと続くわけじゃないんだ。一二時には解けてしまうよ。一二時の鐘がなる前に帰っておいで。あと誰かにダンスを誘われたら断るんじゃないよ？絶対だからな？絶対断るなよ！」~

=「わかったわ、魔法使いさん。どうもありがとう！」~
=
=シンデレラはそう言うと、かぼちゃの馬車に乗ってお城へと出発しました。それを見届けた魔法使いは深い溜息をついて言いました。~
=
=「あーーー、あの呪文噛まずに言えてよかったー。間違えてたらシンデレラカエルになるとこだった。てか疲れたわ。タマネギくらい食え。まあ、でもこれでシンデレラがあの王子と結ばれてくれたらいろいろ助かるんだけどなー。あの王子マジないわ。帰ろ」~

=
=「まあ、ここがお城なのね！」~

=目の前にあるお城を見るとシンデレラは目を輝かせました。~
=
=「シンデレラ、僕らにかけられた魔法は12時になったら解けてしまうからね。
=時間を忘れてはいけないよ♪｢(＾ω＾)┘三└(＾ω＾)｣♪」~

=
=その中で行われている舞踏会と、何よりもおいしい料理に意識を奪われているのでした。~
=
=「いってきます！帰りもお願いね、馬さんたち！」~
=
=そういってシンデレラはお城に入っていくのでした。~
=
=
=

=そのころお城では、継母と姉たちが優雅に舞踏会を楽しんでいました。美しい姉たちは周りの視線を独占し、
=継母もうれしそうにその様子を見つめます。~
=
=「お嬢様方は本当にお美しいですわ。あなた様そっくり」~
=
=「まあ、そんなことございませんわ。まだまだ品格が…おほほ」~
=
=周りからの賛辞も継母は完璧に対応します。さすが継母。猫かぶりでコンピテンシー向上。~
=
=「あそこまで美しいお嬢様方なら、きっと王子様のお目にも留まるわ！」
=
=その一言に継母はピクリと反応しました。なぜなら、王子様についてよい噂を聞かないからです。~
=
=継母はシンデレラを除いた自分の娘には自信を持っていましたし、必ず良いところに嫁がせる気まんまんでしたが、~
=
=直感的に王子様のところには嫁がせてはいけないと思っていました。俗にいう女の勘です。
=そして今日はその王子様を実際に見定めるために舞踏会に行ったのでした。~
=
=「あらあら、ぜひとも王子様にお会いしたいものですわ、おほほほ…」~
=
=継母はそう答えると、まわりがざわつき始めたのに気が付きました。娘たちに何事か問いかけると、
=王子様がお目見えになるそうですと言いました。~
=
=すると王国のラッパ隊が現れ、レッドカーペットが舞踏会の会場の中心に引かれました。~
=
=「王子様がいらっしゃいますわ！」~
=
=周りの期待が高まり、玉座にお座りになる国王様、お妃様もにこやかにされていらっしゃいます。
=
=そしてラッパ隊の演奏が終わると、舞踏会の会場のドアが大きく開き、
=
=奥から皇太子ちゃら様がこれでもかというぐらいのドヤ顔をして現れました。~
=
=舞踏会に集まった方々はみなお辞儀をします。ちゃら王子はレッドカーペットを堂々と進み、玉座の元まで向かいます。~
=
=その後ろには王子が一番信頼する従者、暇ぽんが続きます。王子は集まった貴族たちの間を通り抜け玉座の前に立ち、
=
=国王様、お妃様にご挨拶なさいました。~
=
=「王子、今日の舞踏祭はお前の嫁さがしも兼ねておる。わが妻のようによき娘を探してくるのだ」~
=
=「あなたと一緒にこれからを生きていくことになる人です。いつものようなちゃらく見える態度は慎み、真剣に探すのですよ」~
=
=国王様とお妃様は王子の婚約が心配で心配で、今回の舞踏祭をお開きになられたのでした。
=
=そんなお二人の心配もつゆ知らず、王子は~
=
=「ご安心ください！今日こそ素敵で可憐な私の運命の娘を探し出して見せます！そのためにも多くの娘と話さなければっ！」~

=

=
=王子に娘はやれないとお思いになられていました。いくら王族とはいえ、この王子に嫁がせるくらいなら
=
=公爵様に嫁がせた方がずっといい、と。いろいろ考えを巡らせていた継母は娘たちに公爵様の元へご挨拶に行くように命じました。

=そんなシンデレラを目にとめたのは、従者の暇ぽんでした。がつがつお城の料理を口に運んでいますが、

=
=従者はアニメを三日三晩連続で一緒に見させられたり、ナンパのお供に駆り出されたりして嫌でも覚えてしまったのでした。~

=「おい、お前冷たいぞ！俺とお前はズッ友だろ？」~

=「（め ん ど く せ え）あーはいはい、そーっすね。早く声かけてくればいいんじゃないっすか。」~

=ちゃら王子はシンデレラの乗った馬車が遠くへ行くのを見守るだけでした。遠い目をする王子に
=
=後ろから追いかけてきた従者暇ぽんが追いつきました。~
=
=「さっきの娘はどうしました!?」~
=
=「ああ、あの娘は消えってしまった…」~
=~
=遠い目をする王子を見て悟った従者は王子の肩をぽん、と叩きました。どう慰めたものかと言葉を探していると~

=ガラスの靴が片方だけ落ちているのに気が付きました。~
=~
=
=「王子！これはさっきの娘が落としていったものでは？この靴がっぴったりはまる娘を探し出しましょう、ね！~
=

+シンデレラは迷いました。王子と過ごした時間は楽しかったのですが、結婚となるとまた別です。
+
+シンデレラは結婚はもっと時間をかけて考えたいと思いました。
+
+しかし、いくらシンデレラがそう思っていてもそうさせてくれないのがシンデレラの人生でした。継母がシンデレラにすごんできたのです。
+
+「シンデレラ、あなたもあの日“一緒に”舞踏会に行ったでしょう？あなたもあの靴を履いてごらんなさいな。うふふふ」
+
+継母の後ろに黒いオーラが見えたシンデレラはもう逆らえません。ここで逆らったら玉ねぎを何個食べさせられるのかわかったものじゃないからです。
+
+もう玉ねぎだけしか食べられない一週間を過ごしたくないシンデレラは悲しい顔をしてうなずきました。しかし…。
+
+「おや、シンデレラ。なんか靴を履くのが嫌そうな顔をしていないかい？」
+
+お父さんがシンデレラの様子に気づいたのです。神か。
+
+「シンデレラ、お前が履きたくないのなら無理に履く必要はないんだぞ。王子様には申し訳ないが、父さんと（継）母さんが適当に言っておくから」
+
+「ちょっと、あなた何をっ…」
+
+「君はこんな顔をしたシンデレラにあの靴を履くように言うのかい？僕はそれは許せないな…」
+
+「（うっ……いつになくこの人がいるこの状況は完全に不利だわ…）で、でも王子様はどうするおつもりですの？」
+
+「シンデレラが履いてみたくないというなら、僕が王子とお話しさせていただいて、申し訳ないけど帰っていただくよ」
+
+そういうなりお父さんは王子のところへ行き、説得をして王子を帰らせてしましました。これにはシンデレラも継母も姉たちもびっくりです。
+
+お父さん実はできる人だったか…。
+
+「シンデレラ、君の人生は君のものだよ。君の好きなように生きるといい」
+
+お父さんはそういって微笑みました。シンデレラのお父さんは影は薄いし、あんな継母と結婚してしまうような人ですが、
+
+シンデレラの幸せを一番に願う素敵な人だったのです。
